Есть несколько способов связи с администрацией:

### 1. Через ТГ-чат (рекомендуется)

Переходим по [ссылке](https://chat.florestmc.ru) (если она не работает - попробуйте [эту ссылку](https://t.me/florestchatminecraft)) и в Telegram описываем нашу проблему.\
Этот способ рекомендуется, потому что здесь можно быстрее других способов получить помощь (в ТГ-чате есть не только администрация, но и другие игроки).

### 2. Через ТГ-канал

Этот способ подойдёт, если вам нужен ответ именно от администрацией.\
Для начала переходим по [ссылке](https://channel.florestmc.ru), затем нажимаем на значок сообщения (см. фото ниже) и там описываем нашу проблему.

### 3. Через почту

Если вы можете связываться только через почту (актуально при блокировке мобильного интернета), можете использовать этот способ.\
Заходим в почтовый клиент ([Яндекс Почта](https://mail.yandex.ru), например), в поле "**Кому**" пишем адрес `support@florestmc.ru`, в поле "**Тема**" пишем краткий смысл вашей проблемы и в большом поле уже описываем нашу проблему.
